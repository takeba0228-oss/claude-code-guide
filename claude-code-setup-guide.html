はじめに＋用語の解説

Claude Code = 日本語で「こういうの作って」と話しかけるだけで、AIがコードを自動で書いてくれるツール。
CMUX = Claude Code を快適に使うための Mac 専用アプリ（無料）。

Claude の有料プラン（Pro / Max）月額サブスク。これに入ると Claude Code が使える

このガイドに出てくる言葉

ターミナル
Mac に最初から入っている「黒い画面」のアプリ。文字で命令を出す道具。Launchpad で「ターミナル」と検索すると出てきます。
コマンド
ターミナルに打ち込む命令文。このガイドではコピペするだけなので暗記は不要。
コピペの方法
Command + C でコピー → Command + V で貼り付け。ターミナルでも同じ操作でOK。
Enter（エンター）キー
Mac のキーボードでは 「return」 と書いてあるキー。コマンドを実行するときに押します。
Homebrew（ホームブリュー）
Mac にアプリを簡単にインストールする無料ツール。「App Store のコマンド版」。

英語がたくさん流れる → 放置
3〜5分。触らないでください。

「Next steps」の例（この2行を1行ずつコピペ実行）

6
Claude Code をインストールする
Claude Code のアプリを持っていても、ターミナルでのインストールは必要です！

Claude Code のデスクトップアプリ（Mac アプリ）を既にインストールしていても、CMUX から使うにはターミナルでのインストール（このステップ6）が別途必要です。
アプリ版とターミナル版は別物です。必ず以下の手順を実行してください。

6-1. まず Node.js が入っているか確認

Claude Code を動かすには Node.js というソフトが必要です。まず確認しましょう：

コピペ
% node --version

「v20.x.x」「v22.x.x」など数字が出ればOK → 6-2 へ進んでください。

「command not found」と出たら → Node.js をインストールします：

コピペ
% brew install node

2〜3分で完了。node --version で数字が出ればOK。

Node.js がないと「claude not found in PATH」エラーが出ます！

この後 CMUX で claude と打った時に Error: claude not found in PATH と出る原因の多くは Node.js が入っていないことです。必ずこのステップを実行してください。

6-2. Claude Code をインストール

白い文字をコピペ → Enter（return）：

コピペ
% curl -fsSL https://claude.ai/install.sh | bash

1〜2分で完了。確認：claude --version でバージョンが出ればOK。

「command not found」と出たら

ターミナルを左上の赤い×で閉じて、Launchpad からもう一度開き直してください。
それでもダメなら以下を実行：
npm install -g @anthropic-ai/claude-code

1行ずつコピペ

% brew install --cask cmux

次にこれもコピペ：

コピペ
% sudo ln -sf "/Applications/cmux.app/Contents/Resources/bin/cmux" /usr/local/bin/cmux
「sudo」って何？パスワード聞かれた！

sudo（スードゥー）= 管理者として実行。危険じゃありません。Mac のパスワードを入力してください（画面には表示されません）。

確認：cmux --version でバージョンが出ればOK。

ここまでのターミナルはもう閉じてOK！

左上の赤い×で閉じてください。次からは CMUX を使います。

10
困ったとき・エラー対処法

エラー別トラブルシューティング

Q❌ Error: claude not found in PATH
こんな画面が出たとき
cmux
yourname@MacBook-Air ~ % claude
Error: claude not found in PATH
yourname@MacBook-Air ~ %

原因：Claude Code が正しくインストールされていない、または Node.js が入っていない。

解決手順（上から順番に試してください）：

  1. Node.js を確認：node --version と入力。「command not found」なら brew install node を実行
  2. Claude Code を再インストール：curl -fsSL https://claude.ai/install.sh | bash
  3. CMUX を一度閉じて開き直す（左上の赤い×で閉じる → Launchpad から再度開く）
  4. それでもダメなら：npm install -g @anthropic-ai/claude-code
Q❌ OAuth error: Invalid code
こんな画面が出たとき
cmux
OAuth error: Invalid code. Please make sure the full code was copied

Press Enter to retry.

原因：ブラウザでのログイン認証が途中で切れた・タイムアウトした。

解決手順：

  1. Enter（return） を押してリトライ
  2. リトライしてもダメなら、先に Safari / Chrome で claude.ai にログインしておく
  3. その後 CMUX に戻って claude login と入力してやり直す
ポイント

CMUX の内蔵ブラウザは Google パスワードや2段階認証の入力がしにくいことがあります。先に普通のブラウザでログインしておくのがいちばん確実です。

Q⚠️ ログインが切れた / 再ログインを求められた

一定期間（数日〜数週間）でログインが切れることがあります。これは正常な動作です。

解決手順：

cmux
yourname@MacBook-Air ~ % claude login

claude login と打つだけ！ ブラウザが開くので「許可」をクリック。

ログインしやすくするコツ

CMUX 内のブラウザではなく、普通のブラウザ（Safari / Chrome）を使うと Google ログインのパスワード入力や2段階認証がスムーズです。
先に Safari/Chrome で claude.ai にログインしておけば、CMUX からは「許可」ボタンを押すだけで完了します。

Q❌ node: command not found / Node.js が入っていない

Claude Code は Node.js がないと動きません。

コピペ
% brew install node

完了したら node --version で確認 → 数字が出ればOK。
その後 Claude Code を再インストール：curl -fsSL https://claude.ai/install.sh | bash

Q⚠️ brew: command not found

ステップ5の Homebrew インストール後の「Next steps」の2行を実行し忘れています。

1行ずつコピペ → Enter
% echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
% eval "$(/opt/homebrew/bin/brew shellenv)"

実行後、brew --version で確認してください。

コスト削減テクニック（Pro を長持ちさせる）
なぜコスト管理が大事？

Pro プラン（$20/月）には使用量の上限があります。上限に達すると一時的に使えなくなります。
以下のテクニックで同じ料金でもっとたくさん使えるようになります。

すぐできる！7つの節約テクニック

★テクニックやり方節約効果
★★★1つの会話で1つの作業話題を変えるときは /clear でリセット大
★★★/compact を使う会話が長くなったら /compact で圧縮大
★★★具体的に指示する「いい感じに」→「青色で、ボタンは角丸で」大
★★やり直しより修正指示「全部作り直して」→「ここだけ変えて」中
★★ファイル添付で説明省略「この画像みたいに」「このファイル参考に」中
★★先に方針を聞く「作る前にどういう構成にするか教えて」中
★不要なファイル読み込みを避ける必要なフォルダだけで claude を起動小

会話の節約イメージ

悪い例（コスト浪費）
あなた:

あなた:
うーん、違うな。全部作り直して
あなた:
やっぱこれも違う。もう一回全部
→ 3回やり直し = 3倍のコスト消費...
良い例（コスト節約）

Claude:
作成しました。
あなた:
いいね！「料金」のところだけ表形式にして
→ 最初から具体的 + 部分修正 = コスト1/3！

月 $100 の Max 5x にアップグレードすれば使用量が5倍に。claude.ai → Settings → Subscription からいつでも変更可能。

おすすめの使い方・スキル集
スキルって何？

Claude Code には「スキル」という拡張機能があります。特定の作業に特化した「得意技」のようなもの。
スキルなしでも使えますが、知っているだけで作業効率が何倍にもなります。

初心者におすすめ！すぐ使える活用パターン

やりたいことClaude にこう言えばOK
🌐Webサイトを作る「カフェのホームページ作って。メニューと地図を載せたい」
📝文章を書く・直す「このメールの文章、もっとビジネスっぽくして」
📊Excel / データ整理「このCSVファイルを見やすい表にして」
🎨デザイン調整「このページ、スマホで見やすくして」
🔍調べもの・リサーチ「競合のSNS運用について分析して」

覚えておくと便利な「/コマンド」

Claude Code の中で / から始まる文字を打つと、特別な機能が使えます：

コマンド機能こんなとき使う
/compact会話を圧縮して軽くするコスト節約に必須！長い会話の途中で
/clear会話をリセット別の作業を始めるとき
/cost使用量を確認制限が気になるとき
/helpヘルプ表示使い方がわからないとき
/theme見た目を変更色を変えたいとき
/exit終了作業が終わったとき

上級者向け：さらに使いこなすコツ

「CLAUDE.md に書いておいて」
毎回同じ指示をしなくていいように、ルールを記憶させることができます。
例：「日本語で返答して」「ファイルは Desktop に作って」など。一度設定すれば毎回の会話で自動適用。
「前回の続きから」
Claude を閉じて再度開いたとき、作ったファイルはそのまま残っています。「さっき作ったサイトの続きお願い」と言えば、ファイルを読み取って理解してくれます。
画面分割で複数作業
CMUX で Command+D すると画面が分割され、片方で Claude、片方で別の作業ができます。
ファイルをドラッグ＆ドロップ
参考にしたい画像やファイルを CMUX の画面にドラッグ＆ドロップすると、Claude が内容を読み取ってくれます。「この画像みたいなデザインにして」がすぐできます。
最大のコツ

「友達に頼むように話しかける」これだけ。
完璧な指示じゃなくてOK。「ここ変えて」「もうちょっと派手に」「やっぱ元に戻して」── ぜんぶ伝わります。

よくある質問
日本語で使える？
はい。完全対応。英語不要。
プログラミング知識なくても使える？
はい。「Webサイト作って」でOK。細かい微調整には基礎知識があると便利。
普通のターミナルでも使える？
使えます。CMUX の方が画面分割やチーム機能が使えて便利。
Windows でも使える？
Claude Code は可。CMUX は Mac 専用。Windows は PowerShell か VS Code で。
APIキーは必要？
不要。Pro/Max アカウントでログインするだけ。
データ漏れない？
PC上で動作。Pro/Max のデータはAI学習に使われません。
閉じちゃった！データ消えた？
ファイルは消えません。会話履歴だけリセット。「前回の続きから」と言えばOK。
固まって動かない！
Ctrl+C 2回で強制終了 → claude で再起動。
「claude not found in PATH」って出る！
Node.js が入っていない可能性大。CMUX で brew install node → curl -fsSL https://claude.ai/install.sh | bash で解決。詳しくはセクション10参照。
ログインが切れた！
CMUX で claude login と打てば再ログインできます。先に Safari/Chrome で claude.ai にログインしておくとスムーズ。
テーマ選択・ログイン方法で何を選ぶ？
テーマは 「1. Auto」、ログイン方法は 「1. Claude account with subscription」 を選んでください。
料金は日本円じゃないの？
米ドル（USD）決済。カード会社が自動で円換算します。
